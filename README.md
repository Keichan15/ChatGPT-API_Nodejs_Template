# ChatGPT-API & Node.js Template

## Usage
1. Installed Library.

```terminal:terminal
$ npm install --save openai
```
2. Change your APIKey.

```javascript:index.js```
const openai = new OpenAI({
  // change your APIKey
  apiKey: "sk-xxxxxxxxxxxxxxx",
});

3. Run this Application.
```terminal:terminal
$ node index.js
```
